# Eli's English Adventures - Welcome Screen

A beautiful Flutter welcome screen for a kids' English learning app featuring:

## 🎨 Design Features

- **Theme**: Soft, cheerful, and child-friendly
- **Colors**: Sky Blue (#87CEEB), Warm Orange (#FFA500), White background
- **Font**: Nunito (rounded, playful from Google Fonts)
- **Style**: Vibrant with soft shadows and subtle bounce animations

## 🧩 Screen Components

- **Background**: Bright jungle classroom theme with gradient
- **Animated Mascot**: "Eli the Elephant" with waving animation
- **App Title**: "Eli's English Adventures" with orange accent
- **Subtitle**: "Learn English with fun games and adventures!"
- **Start Learning Button**: Big rounded orange button
- **Sign In Option**: Link for existing users
- **Floating Elements**: Letters A, B, C and stars with soft motion

## 📁 Project Structure

```
lib/
├── main.dart                    # App entry point
├── screens/
│   └── welcome_screen.dart     # Welcome screen implementation
└── theme/
    ├── app_colors.dart         # Color palette and gradients
    └── app_text_styles.dart    # Text styles with Nunito font
```

## 🚀 Getting Started

1. **Install dependencies**:

   ```bash
   flutter pub get
   ```

2. **Run the app**:
   ```bash
   flutter run
   ```
